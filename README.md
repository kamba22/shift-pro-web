# Shift PRO - Frontend 🕒

Shift PRO is a sleek, professional workforce management dashboard designed for tracking employee shifts, monitoring real-time status, and managing work history.

##  Features
- **Real-time Clocking:** Employees can clock in/out with a live elapsed timer.
- **Shift Summaries:** Mandatory work reporting upon clocking out.
- **Admin Monitor:** A dedicated view for administrators to track active staff and total hours.
- **Responsive UI:** A clean, modern interface built with a focus on usability and professional aesthetics.

## Tech Stack
- **React.js** (Functional Components & Hooks)
- **CSS3** (Custom modular styling)
- **Supabase** (Authentication and Real-time Database)
- **Lucide React** (Iconography)
